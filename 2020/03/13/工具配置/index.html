default config

1.

  1. 在生成的文件中搜索： 把文件夹路径填进去，去掉#号，保存
  2. 在开始菜单找到“Jupyte Notebook”快捷键，鼠标右击 – 更多 – 打开文件位置
  3. 找到对应的“Jupyte Notebook”快捷图标，鼠标右击 – 属性 – 目标，去掉后面的 “%USERPROFILE%/“（很重要），然后点击“应用”，“确定”　　
  4. 重新启动Jupyte Notebook即可

VScode

debug

  • 单步跳过：一步一步的执行，循环内也是一步一步的执行，调用函数时不会进入函数内部；
  • 单步调试：一步一步的执行，循环内也是一步一步的执行，调用函数时会进入函数内部；
  • 单步跳出：执行这个断点和下一个断点之间的语句，要是这是最后一个断点，那么完成程序的执行；

R in vscode

step:

  1. 自己安装R和Rtools
  2. 安装包languageserver
  3. 安装R language support插件
  4. 使用pip install radian，安装好R console
  5. 在vsc中，将r.term.window设置成radian.exe的路径
  6. 勾选always use active terminal
  7. 每次写R时，要手动在vsc的终端中启动radian

话说VS code可以把=恰当的转化为<-，有点优秀啊（只转化该转化的）
