Git学习教程

git

分布式控制，每个人把文件保存在本地电脑上，修改后提交给别人。
所以每一个设备需要自报家门：

这样一来，这台电脑所有的仓库均使用这一个ID

把一个文件夹变成一个git可以管理的仓库，会生成 .git文件夹(windows系统下默认隐藏)。

git只可以管理文本文件，word、ppt啥的别想了

千万不要使⽤Windows⾃带的记事本编辑任何⽂本⽂件。原因是Microsoft开发记事本的团队使用了一个个非常弱智的⾏为来保存UTF-8编码的⽂件，他们⾃作聪明地在每个⽂件开头添加了0xefbbbf（⼗六进制）的字符。建议你下载Notepad++代替记事本，不但功能强⼤，⽽且免费！记得把Notepad++的默认编码设置为UTF-8 without BOM即可

添加⽂件到Git仓库，分两步：

  • 第⼀步，使⽤命令git add ，注意，可反复多次使⽤，添加多个⽂件，命令成功后没有提示；
  • 第⼆步，使⽤命令git commit，提交文件到仓库完成，有提示。
