Hexo+Butterfly+Github Pages配置个人博客网站

Posted on 2020-07-11 Edited on 2020-10-25 In 工具 Valine:
Symbols count in article: 2.8k Reading time ≈ 3 mins.

之前我也用Jekyll配置过博客，但是访问速度太慢，我又转向了Hexo，实践证明，Hexo还是很快的。我又换到Next了

前期准备

  1. install git and node.js(better use npm taobao mirror or cnpm)
  2. config your github account infomation in local;
  3. create a new reposity named username.github.io;
  4. open C:\Users\用户名\.ssh\id_rsa.pub and copy the whole text;
  • login Github in chrome
  • setting —-> SSH and GPG keys —-> new SSH KEY —-> paste
