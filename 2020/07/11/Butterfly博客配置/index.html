Hexo+Butterfly+Github Pages配置个人博客网站

Posted on 2020-07-11 Edited on 2020-10-25 In 工具 Valine: Symbols count in article: 2.8k

之前我也用Jekyll配置过博客，但是访问速度太慢，我又转向了Hexo，实践证明，Hexo还是很快的。我又换到Next了

前期准备

  1. install git and node.js(better use npm taobao mirror or cnpm)
  2. config your github account infomation in local;
  3. create a new reposity named username.github.io;
  4. open C:\Users\用户名\.ssh\id_rsa.pub and copy the whole text;
  • login Github in chrome
  • setting —-> SSH and GPG keys —-> new SSH KEY —-> paste

  1. install Hexo in local; 安装完之后，它默认使用的是landscape主题(主题都在themes这一文件夹下)

  3. 安装相关依赖

可以使用这样的方法来连续执行多条CMD命令：
创建 hexo.bat 文件，写入以下命令，双击运行即可

d: && cd Blog && hexo clean && hexo g && hexo d && hexo s

自定义Butterfly

修改右上角为中文

  1. 修改根目录下的_config.yml中的语言
  2. 修改 theme/Butterfly文件夹下的_config.yml
  3. 保存
  4. 预览

以创建tags页为例，其它同理。

  2. 点击存储，创建类： Comment, Counter;
  3. 设置，安全中心，只打开数据存储；
  4. 设置，应用keys，复制AppID，AppKey;

页面配色

  1. 主题配色在_config.yml中修改，参见官方文档
  2. 主页上的划分的一个一个card，在这里修改
